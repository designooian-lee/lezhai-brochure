# 乐宅.Life 图册选款平台

手机优先的电子图册分类、站内阅读与后台管理平台。当前支持云展网、goootu 和 FLBOOK。

## 本地使用

双击根目录的 `一键启动.bat`。首次启动会初始化 PostgreSQL、创建数据库、生成随机后台密码、导入示例图册并打开客户页面。

- 前台：http://127.0.0.1:8080/brochure/
- 后台：http://127.0.0.1:8080/brochure/admin/login
- 健康检查：http://127.0.0.1:8080/brochure/health
- 本地登录凭据：`storage/runtime/local-login.txt`

双击 `停止平台.bat` 可停止本项目启动的网页服务。

后台“数据管理”可导出或恢复 ZIP 备份。导出时会询问是否包含本地高清页面；选择不包含可明显减小备份体积。

前端每本图册提供收藏和分享。收藏仅保存在客户当前浏览器；分享链接形如 `/brochure/catalog/{图册编号}`，好友打开后会自动进入对应图册。
